/**
     *当任务执行完成后，返回执行的结果，该方法和`Feture.get()`不同的地方时，其抛出的异常是`RuntimeException`和`Error`。
     *此外，也不会抛出`InterruptedException`。
     */
    public final V join() {
        if (doJoin() != NORMAL) // 任务没有正常完成
            return reportResult(); //处理非正常情况
        else
            return getRawResult(); // 返回结果
    }
